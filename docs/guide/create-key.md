# 创建 Key

注册登录后，进入控制台创建 API Key。

## 创建步骤

1. 点击左侧菜单「我的 Keys」
2. 点击「创建」按钮
3. 选择需要的路由（根据你要使用的模型选择）
4. 保存并复制生成的 Key

::: warning ⚠️ 注意
Key 只会显示一次，请务必保存好。如果丢失，需要重新创建。
:::

## 路由说明

选择 **codex 路由**可以在 Claude Code 等工具中使用 GPT 系列模型（opus 对应 gpt-5.2）。

## 安全建议

::: danger 🔐 安全提醒
- **不要**将 Key 分享给他人
- **不要**将 Key 提交到代码仓库
- **不要**在公开场合展示 Key
- 使用环境变量或配置文件存储 Key
:::
