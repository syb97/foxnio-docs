# 欢迎使用 FOXNIO

FOXNIO 是一家 Claude / GPT 系列模型的 API 中转服务站，支持 Claude 和 GPT 最新模型，国内直连、无需魔法。

群佬们也在出售 **GPT Plus** 和 **Pro**，价格非常实惠，有意可联系群主。

| 站点 | 地址 | 说明 |
|------|------|------|
| 公益站 | [www.foxnio.com](https://www.foxnio.com) | 免费，支持 GPT-5.2 及之前的模型 |
| API 付费站 | [api.foxnio.com](https://api.foxnio.com) | 充值/商业用途，支持更多高级模型 |

## 快速开始

如果您是第一次使用，建议按照以下步骤操作：

1. [注册账号](/guide/registration) - 获取邀请码并注册
2. [创建 Key](/guide/create-key) - 获取 API 密钥
3. 部署工具 - 选择你需要的工具：
   - [Claude Code 部署](/deploy/claude-code)
   - [CodeX 部署](/deploy/codex)
   - [OpenClaw 配置](/deploy/openclaw)
