# 注册账号

## 获取邀请码

1. 访问 [www.foxnio.com](https://www.foxnio.com/)
2. 进入砍价页面，参与砍价活动
3. 砍价砍到 **20 元以下**，即可联系群主获取邀请码

::: tip 💡 提示
砍价金额低于 20 元后，截图发给群主即可领取邀请码。
:::

## 注册步骤

1. 使用获取到的邀请码，在 [www.foxnio.com](https://www.foxnio.com/) 完成注册
2. 设置账号密码
3. 登录账号
